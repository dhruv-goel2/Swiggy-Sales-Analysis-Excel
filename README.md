# Swiggy Sales Analysis Dashboard (Excel)

## Project Overview

This project analyzes Swiggy food delivery data using Microsoft Excel. The dashboard provides insights into sales performance, customer ratings, food categories, city-wise trends, and pricing segments.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Formulas

## Dataset

The dataset contains over 197,000 food delivery records across multiple cities and states.

## Key KPIs

* Total Revenue
* Average Rating
* Average Order Value (AOV)
* Total Orders
* Category Performance
* City-wise Analysis

## Dashboard Features

* Interactive Slicers
* Food Category Analysis
* Price Segment Distribution
* Weekday vs Weekend Analysis
* City-wise Performance Tracking
* Restaurant Performance Insights

## Key Insights

* Identified top-performing food categories.
* Analyzed customer ratings across restaurants.
* Compared weekday and weekend ordering patterns.
* Evaluated pricing segment trends.
